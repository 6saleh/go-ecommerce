# Product Roadmap – Go Fiber E-Commerce

## 1. Vision & Tech Stack
- **Problem:** بناء متجر إلكتروني متوسط قابل للبيع بسهولة.
- **Solution:** منصة بسيطة وحديثة تعتمد Go + Fiber + SQLite3.
- **Tech Stack:** Go، Fiber، SQLite3، HTML/CSS/JS.
- **Constraints:** واجهات بسيطة، بناء وحدات فقط، تصميم مألوف.

## 2. Functional Modules
| Priority | Module | Rationale | Description |
|---------|---------|-----------|-------------|
| 1 | Core Foundation | Project bootstrapping | Folder structure، config، DB init |
| 2 | Auth | Basic requirement | Signup, login, sessions |
| 3 | Products | Core store feature | CRUD + images |
| 4 | Cart | User shopping workflow | Add/remove items |
| 5 | Orders | Purchase flow | Order creation + status |
| 6 | Dashboard | Admin control | Products, orders mgmt |
